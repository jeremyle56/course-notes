              \item \(\Omega_1 = \{\fx = (x_1, x_2, x_3) \in \bR^3: x_1 + x_2 + x_3 = 1, x_i \geq 0, i = 1, 2, 3\}\). We see that \(n = 3\) and \(m = 1\), then hence an extreme point has \(n - m = 2\) entries are zero. So candidates for extreme points are
                    \[\fx_1^* = \begin{pmatrix}
                            1 & 0 & 0
                        \end{pmatrix}, \quad \fx_2^* = \begin{pmatrix}
                            0 & 1 & 0
                        \end{pmatrix}, \quad \fx_3^* = \begin{pmatrix}
                            0 & 0 & 1

                    Now \(x_1 + x_2 + x_3 = 1 \iff \fa_1^T \fx = b_i\) with \(\fa_1 = \begin{pmatrix}
                        1 & 1 & 1
                    \end{pmatrix}^T\) and \(b_i = 1\). So \(A = \fa_1^T = \begin{pmatrix}
                        1 & 1 & 1
                    \end{pmatrix}\). For \(\fx_1^*\), as only the first component is non-zero and \(\{A_1\}\) is linearly independent then \(\fx_1^*\) is an extreme point of \(\Omega_1\). Similarly \(\fx_2^*\) and \(\fx_3^*\) are also extreme points of \(\Omega_1\).
              \item \(\Omega_2 = \{\fx = (x_1, \dots, x_n) \in \bR^n : \sum_{i = 1}^n x_i = 2, x_i \geq 0, i = 1, \dots, n \}\). Similarly \(m = 1\), so there are \(n - 1\) zero entries in the extreme points. Then from above, it follows that each \(2\fe_i = \begin{pmatrix}
                        0 & \cdots & 2 & \cdots & 0
                    \end{pmatrix} \in \bR^n\) is an extreme point of \(\Omega_2\).
              \item \(\Omega_3 = \{\fx = (x_1, x_2, x_3, x_4) \in \bR^4 : x_1 + x_2 + x_3 = 1, x_2 + x_4 = 2, x_i \geq 0, i = 1, \dots, 4\}\). Note that \(n = 4\) and \(m = 2\), so extreme points have \(n - m = 2\) entries are zero. Then note that
                    \begin{align*}
                        x_1 + x_2 + x_3 = 1 & \iff \fa_1^T \fx = 1 \\
                        x_2 + x_4 = 2       & \iff \fa_2^T \fx = 1
                    \end{align*}
                    where \(\fa_1 = \begin{pmatrix}

                        0 & 1 & 0 & 1
                    \end{pmatrix}^T\). So, \(A = \begin{pmatrix}
                        \fa_1^T \\ \fa_2^T
                    \end{pmatrix} = \begin{pmatrix}
                        1 & 1 & 1 & 0 \\
                        0 & 1 & 0 & 1
                    \end{pmatrix}\).

                        \item If \(x_1 = x_2 = 0\), then \(x_3 = 1, x_4 = 2\). As \(\{A_3, A_4\}\) are linearly independent, \(\begin{pmatrix}

                              \end{pmatrix}\) is an extreme point.

                                  0 & 1 & 0 & 1
                              \end{pmatrix}\) is an extreme point.
                        \item If \(x_1 = x_4 = 0\), then \(x_2 = 2\) and \(x_3 = -1\) which is not in the feasible region.
                        \item If \(x_2 = x_3 = 0\), then \(x_1 = 1, x_4 = 2\). As \(\{A_1, A_4\}\) are linearly independent, \(\begin{pmatrix}
                                  1 & 0 & 0 & 2
                              \end{pmatrix}\) is an extreme point.
                        \item If \(x_2 = x_4 = 0\), then \(x_2 + x_4 \neq 2\) which is impossible.
                        \item If \(x_3 = x_4 = 0\), then \(x_2 =2, x_1 = -1\) which is not in the feasible region.

                    \begin{minipage}[t]{0.45\textwidth}
                        \textbf{Dual Problem}
                        \begin{align*}
                            (D) \text{ maximize} \quad & y_1 + y_2 + y_3                       \\
                            \text{subject to} \quad    & \begin{pmatrix}
                                -y_1 & 1    & 0    \\
                                1    & -y_2 & 0    \\
                                0    & 0    & -y_3
                            \end{pmatrix} \succeq 0.
                        \end{align*}
                    \end{minipage}
              \item The optimal value of \((D) = -2\) as the primal problem and dual have non-empty feasible regions, and the interior point condition is satisfied. By the strong duality theorem for conic programs, this implies that the optimal value of \((D)\) is the optimal value of \((P)\).
          \end{enumerate}
    \item \begin{enumerate}[label=(\arabic*)]
              \item Let \(C_1 = \SOC^3 = \{ \fx \in \bR^2: x_1 \geq \sqrt{x_2^2 + x_3^2} \}\) and \(C_2 = \bR_+ = \{x \in \bR : x \geq 0\}\). Then \(K = C_1 \times C_2\). Now,
                    \begin{align*}
                        K^* & = (C_1 \times C_2)^* \\
                            & = C_1^* \times C_2^*
                            & = C_1 \times C_2     \\
                            & = K.
                    \end{align*}
                    as \(C_1^* = (\SOC^3)^* = \SOC^3 = C_1\) and \(C_2^* = (\bR_+)^* = \bR_+ = C_2\). So \(K\) is a self-dual cone.
              \item    \begin{align*}
                        \text{maximize} \quad   & 1 \cdot y_1 + 0 \cdot y_2               \\
                        \text{subject to} \quad & \begin{pmatrix}
                            -y_2 \\ -1 - y_1 \\ -y_2 \\ -y_1
                        \end{pmatrix} \in K^* = K.
                    \end{align*}
              \item For the primal problem, take any \(\fx\) which is feasible. Then \(x_1 + x_3 = 0\) and \(x_1 \geq \sqrt{x_2^2 + x_3^2}\) so \(x_1^2 \geq x_2^2 + x_3^2 = x_2^2 + x_1^2 \implies x_2 = 0\). So the optimal value of the primal problem is 0. \\

                    Now take any feasible point for the dual. Since \(\begin{pmatrix}
                        -y_2 \\ -1 - y_1 \\ -y_2 \\ -y_1
                    \end{pmatrix} \in \SOC^3\) we have
                    \begin{align*}
                        -y_2 \geq \sqrt{(-1 - y_1)^2 + (-y_2)^2} \\
                        (-y_2)^2 \geq (-1 - y_1)^2 + (-y)^2      \\
                        y_1 = -1.
                    \end{align*}
                    So optimal value is -1 since for any feasible point \(y_1 = -1\). Thus duality gap is optimal value of primal - optimal value of dual gives \(0 - (-1) = 1\).
              \item We note that the primal problem does not satisfy the interior point condition which causes the duality gap to be non-zero. Indeed the interior point condition requires that the system has a solution. However if \(\fx \in \operatorname{int} K\), then \(x_1 > \sqrt{x_2^2 + x_3^2} \geq |x_3|\) which contradicts with \(x_1 + x_3 = 0\).
          \end{enumerate}
\end{enumerate}
